Test are written in [YAML](https://yaml.org/) format called **buildspecs** that is used to describe test configuration. buildtest will translate the configuration into a valid test script that can be executed on the system. The framework is implemented in Python and uses [JSON Schema](https://json-schema.org/) for validation of buildspecs. Learn more [here](/about/). {% endcapture %} {{ markdown | markdownify }}
